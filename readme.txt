Upload video files (MP4) and display in frontend for your users to view and download. Videos can be divided into different categories. Tags are also supported. Video search widget is available for video search.

== Description ==

This plugin is for those who want to upload their own videos on the site and make them available for users to view and download. The videos should be in MP4 format.

* Upload videos. Option to upload normal and HD videos is available.
* Users can download normal or HD videos as they like.
* Create categories for videos. So the videos will be organized.
* Create tags for videos.
* Display videos in front-end.
* User can view & download videos from front-end.
* 3 widgets are available with the plugin. 1) Video search widget 2) Video categories widget 3) Video tag cloud widget.

== Installation ==

1. Upload `videos.zip` to the `/wp-content/plugins/` directory.
2. Activate the `Videos` plugin through the 'Plugins' menu in WordPress.
3. Go to `Videos` and create videos. Add video title, description, video file (MP4) to create video.
4. Add categories and tags for videos if you want. That's it.
5. Now visit http://www.your-site.com/video/ and you will see the videos.

== Frequently Asked Questions ==

= Getting 404 not found on videos page (http://www.example.com/video/ ) =
In admin panel go to <b>Settings -> Permalinks</b>
Here click on save changes. You don't have to change anything just save the permalink settings and the issue should be resolved.
